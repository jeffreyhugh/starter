## [Daisy UI Themes](https://daisyui.com/docs/themes) and [`next-themes`](https://www.npmjs.com/package/next-themes)

Themes need to be updated in `constant/colors.ts` and in `tailwind.config.js`.
Once updated there, additional colors (content and focused background) are generated automatically.

`next-themes` handles theme changing.
In its current configuration, it switches between light and dark mode; however, setting up more themes should be very easy.

## Custom Anchor Tags

No longer do you need to worry about `import Link from next/link` or `target="_blank"`.
Just use an `<A>` tag.

Internal links (i.e. routes that use CSR) will be automatically wrapped in a `<Link>` tag.
External links will open in a new tab.

(It works in MDX, too!)

```jsx
() => (Go to <A href="https://github.com">GitHub</A> or go <A href="/">Home</A>);
```

```md
Go to [GitHub](https://github.com) or go [home](/)
```

Go to [GitHub](https://github.com) or go [home](/)
